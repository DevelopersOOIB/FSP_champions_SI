## Таск Strange file 2

## Описание: 
форензики принесли какой-то странный файл, не знают что с ним делать, возьмешься ?
Файл qcow2 это файл диска в qemu, надо найти как его открыть и почитать содержимое файла hello.txt содержащее флаг 
## Решение 
Cмонтировать диск в систему в линуксе через mount -o loop 123.qcow2 /tmp/mount_point или открыть через 7zip и грепнуть в файле строку, содержащую флаг grep -ri 'flag{'
